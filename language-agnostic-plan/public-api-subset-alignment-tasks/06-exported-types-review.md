# Задача 06. Проверить остальные экспортируемые типы на ту же проблему

## Цель

После исправления основных проблемных типов пройтись по всем остальным публичным экспортам и найти такие же расхождения по форме API.

## Что нужно проверить

1. Все символы из `aspose.note.__all__`.
2. Все публичные свойства и методы у экспортируемых типов.
3. Наличие Python-only членов на .NET-named compatibility-типах.
4. Наличие missing-but-expected членов там, где тип уже считается частью subset.

## Особое внимание

- `Document`
- `Page`
- `TextRun`
- `Image`
- `Title`
- `Table`, `TableRow`, `TableCell`
- `LoadOptions`
- исключения

## Критерии приёмки

- по каждому экспортируемому типу есть решение: соответствует, требует правки или должен быть скрыт
- не осталось крупных неучтённых shape-расхождений после исправления P0
